# go-app-admin

使用 [go-app](https://go-app.dev/) 做为go前端框架的 H5 手机端 admin 项目

## dev

```bash
# dependents
go mod tidy
# or
go get "github.com/maxence-charriere/go-app/v9/pkg/app"
```

## build & run

```bash
make build
make run
```
